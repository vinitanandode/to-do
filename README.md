### Features

- Sign in with Google
- Sign in with Email
- Email Verification
- Add, Edit and Delete tasks
- Mark tasks as important
- View your pending and completed tasks
- Responsive UI

### Technology Stack

* React JS
* Redux
* Styled Components
* Firebase
* HTML
* CSS
* JavaScript

## Run Locally

Clone the project

```bash
  git clone https://github.com/vinitanandode/to-do.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Deployment

To deploy this project run

```bash
  npm run deploy
```
